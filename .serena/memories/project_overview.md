# Watchtower Project Overview

**Watchtower** is a premium, high-performance domain and service monitoring desktop application built with **Tauri**, **Rust**, and **React**. It provides real-time health checks, latency analytics, and historical logging for your digital infrastructure.

## Key Features
- **Real-time Monitoring**: Instant HTTP/HTTPS status probes.
- **Premium UI/UX**: Dark/light mode, glassmorphism, smooth animations.
- **Historical Logs**: Local NDJSON logging.
- **Global Loading System**: Interactive navigation and data sync.
- **Smart Grouping**: Domain organization.
- **High Performance**: Rust backend + Tauri bridge.

## Tech Stack
- **Backend**: Rust + Tauri v2
- **Frontend**: React 19 + TypeScript + Tailwind CSS v4 + TanStack Router
- **Styling**: Tailwind CSS v4
- **Tooling**: Biome (Lint/Format), Husky + lint-staged
- **Package Manager**: pnpm (v10.26.0)
